★ 퍼뜩퍼뜩 ★

안녕하세요.  퍼뜩퍼뜩을 이용해 주셔서 감사합니다.

퍼뜩퍼뜩은 본인이 직접 알람을 설정해서 효과적인 쪽잠을 자고 싶은 분들에게 좋은 프로그램입니다.
기본 설정은 프로그램에 내장되어 있는 알람만 들을 수 있는 제한이 있고
커스텀 설정은 본인이 직접 url을 삽입하여 본인이 설정한 알람을 들을 수 있습니다.

⊙ 퍼뜩퍼뜩을 이용할 때 알아야 할 사항 ⊙
1. 음악을 중간에 끌 수 없습니다. (본인이 직접 강종을 해야합니다... 개발 중에 있으니 양해부탁드립니다.)
2. 프로그램 썸네일처럼 폰트가 적용이 안됩니다. (여러번 시도 끝에 실패... 이것도 개발 중에 있습니다. 양해 부탁)

!!!!!가장 중요!!!!!!!!!!!!

3. 시간을 오래 설정하면 화면이 절전 모드에 들어가서 화면이 꺼집니다. 화면 안 꺼지게 설정 부탁드립니다.
